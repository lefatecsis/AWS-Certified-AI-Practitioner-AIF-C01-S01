Questão 1:

Explicação:

A resposta correta é B. Gráficos de dependência parcial (PDPs). Transparência e explicabilidade em IA/ML referem-se à capacidade de entender como um modelo faz suas previsões. Gráficos de Dependência Parcial (PDPs) contribuem diretamente para a explicabilidade. Um PDP visualiza o efeito marginal de um ou dois recursos no resultado previsto de um modelo de machine learning. Questão 3:

Resposta Correta: A

Explicação:

A resposta correta é A, Árvores de Decisão. Árvores de Decisão são algoritmos de machine learning altamente interpretáveis. Sua estrutura imita um fluxograma, com cada nó interno representando um teste em um atributo (característica do gene, neste caso), cada ramo representando o resultado do teste, e cada nó folha representando um rótulo de classe (uma das 20 categorias de genes). Redes neurais (D) são notoriamente "caixas-pretas" e difíceis de interpretar. Regressão linear (B) é para valores contínuos, e Regressão logística (C) é primariamente para classificação binária.

Questão 7:

Resposta Correta: B

Explicação:

A resposta correta é B. Usar aprendizado por transferência. O aprendizado por transferência é uma técnica de machine learning onde um modelo desenvolvido para uma tarefa é reutilizado como ponto de partida para um modelo em uma segunda tarefa. Isso permite que a empresa aproveite o conhecimento que os modelos pré-treinados (específicos de domínio) já adquiriram, em vez de construir modelos inteiramente novos do zero. Isso reduz significativamente o tempo de treinamento, os recursos computacionais e a quantidade de novos dados rotulados necessários.
